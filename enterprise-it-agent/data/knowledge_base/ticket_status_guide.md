# Ticket Status Guide

Support tickets typically move through these states:

- New: the ticket was created and has not been assigned.
- In Progress: the assigned engineer is working on it.
- Waiting for User: support needs more information from the requester.
- Resolved: the issue appears fixed and is waiting for confirmation.
- Closed: the requester confirmed the result or the SLA window ended.

When a user asks about ticket progress, support should first confirm the ticket ID.
