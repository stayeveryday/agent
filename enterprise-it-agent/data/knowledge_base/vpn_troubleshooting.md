# VPN Troubleshooting

If a user cannot connect to the company VPN, first confirm whether the problem affects all websites or only internal systems.

Basic checks:

- Confirm the user's internet connection is stable.
- Verify the username and password are correct.
- Check whether multi-factor authentication has expired.
- Ask whether the VPN client shows a specific error code.

If the VPN connects but internal systems still fail:

- Confirm the assigned IP address is in the expected range.
- Ask the user to disconnect and reconnect the VPN.
- Clear cached credentials and retry.
- Escalate to network operations if multiple users are affected.
