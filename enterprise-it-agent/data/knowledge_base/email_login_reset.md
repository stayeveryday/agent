# Email Login Reset

If a user cannot sign in to company email, confirm whether the password was changed recently.

Recommended support flow:

- Verify the account name and department.
- Check whether the mailbox is locked.
- Ask the user to try webmail and desktop client separately.
- If password reset is required, follow the account recovery checklist.

Escalate the issue when:

- The account is locked more than twice in one day.
- Multi-factor authentication fails after reset.
- The issue affects multiple users in the same department.
